## 通用

### form
    action="" # 访问的url
    method="POST" # 访问的方法
    enctype="multipart/form-data" # 如果是上传文件的话需要这个

### input
    type="类型" # text, number, radio, checkbox, button, file
    placeholder="默认显示的字符串"
    name="name" # 用来form获取数据
    size=50 # 尺寸
